Mach-II 
==============================================

Mach-II is an open source MVC framework for ColdFusion 
based on the Implicit Invocation design pattern and 
released under the Apache License version 2.0. 

To install this adapter for CacheBox in your Mach-II 
application, place this file in the directory 
/MachII/caching/stragegies/ then edit your MachII.xml 
config file to configure your cache. 

For more information about the Mach-II framework, 
visit the official website at: http://www.mach-ii.com
